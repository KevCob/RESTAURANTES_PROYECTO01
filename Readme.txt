Análisis Estadístico de Restaurantes de La Habana
Este proyecto se centra en la recopilación, análisis y visualización de datos de 200 restaurantes situados en La Habana. Al aprovechar el poder de varias bibliotecas de Python, buscamos comprender mejor las características y tendencias del panorama gastronómico local.

Para alcanzar estos objetivos, utilizaremos las siguientes bibliotecas de Python:

pandas: para la manipulación y análisis de los conjuntos de datos.

plotly: para la creación de gráficos interactivos que faciliten la interpretación visual de los datos.

folium: para la representación geoespacial de los datos, permitiendo mapear la localización de los restaurantes.

ipywidgets: para crear una interfaz interactiva y así facilitar la exploración de datos.

Este análisis puede ser una base útil para futuras investigaciones y aplicaciones prácticas.Al combinar la manipulación de datos con visualizaciones accesibles, esperamos que este proyecto sea una herramienta valiosa y de fácil uso para distintos tipos de usuarios.
